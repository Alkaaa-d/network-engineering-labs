# Lab 1 - Basic LAN Setup

## Objective
Connect two PCs through a switch and verify connectivity.

## Topology
PC0 ---- Switch0 ---- PC1

## IP Configuration

PC0: 192.168.1.10/24

PC1: 192.168.1.20/24

## Verification

Successful ping between both devices.
